Known issues

Issues with the current implementation which may be fixed in the future.

  • robotstxt: The website's robots.txt is not being evaluated. That's because Google, Bing handle it differently.
    Note: At least the DENY (and ALLOW) rules must be implemented for a publication of the crawling framework.
    Note: Google does not respect crawl-delay setting.
  • sitemap: The sitemap parser is provisorical and can only parse sitemaps from disk. That's because the standard for sitemaps is not adhered to by the websites.
  • nonutf8: The original character encoding of a downloaded page is ignored by the framework. Consequently, non-UTF-8 pages are written by the included disk writer as UTF-8. This is a limitation of the framework and will likely not be fixed (see Limitations).
